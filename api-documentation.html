📝 Test Series Management

GET /api/admin/tests Auth Required
Get all test series with details

Update test series price (creates price history)
📂 testSeriesRoutes.js

• newPrice (number, required) - New price for the test
• reason (string, optional) - Reason for price change

Get price change history for a test
📂 testSeriesRoutes.js
